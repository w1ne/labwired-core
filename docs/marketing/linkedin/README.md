# LinkedIn Content Strategy

Carousel-style LinkedIn posts for LabWired simulation platform awareness.
Inspired by Complioty's TARA threat-modeling carousel format (5-slide structure).

## Post Format (Carousel Template)

| Slide | Purpose | Guidelines |
|-------|---------|------------|
| 1 | **Hook** | Bold stat or pain point. Dark background, large text. |
| 2 | **Pipeline** | 5-step visual flow showing how LabWired works. |
| 3 | **Value Props** | 3 concrete benefits in card format. |
| 4 | **Proof/Results** | Real numbers from a demo or case study. |
| 5 | **CTA** | Urgency + call to action (comment/DM/link). |

## Design Notes

- Dark background (#1a1a2e or similar), white/green accent text
- LabWired logo on every slide
- Use monospace or bold sans-serif fonts
- Keep text minimal: max 3 lines per block
- Numbers should be large and prominent (48pt+)
